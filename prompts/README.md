## 💡 使用方法

### 基础使用步骤
1. **浏览分类** - 根据需求选择合适的分类目录
2. **复制提示词** - 选择适合你需求的提示词
3. **个性化调整** - 根据具体场景修改提示词内容
4. **应用到AI平台** - 在任意AI助手中使用

### 高级使用技巧
- **组合使用**: 可以组合多个相关提示词以获得更好的效果
- **迭代优化**: 根据输出效果不断调整提示词
- **角色扮演**: 利用提示词中的角色设定提升回答质量
- **格式控制**: 通过指定输出格式获得结构化的结果

### 使用示例
```
用户: 请扮演一名高级软件架构师，帮我评审以下代码...
AI: [根据提示词扮演架构师角色进行代码评审]
```

## 📖 提示词编写技巧

### 基础原则
1. **明确角色** - 告诉AI扮演什么角色（如：资深架构师）
2. **清晰任务** - 具体描述要完成的任务
3. **详细约束** - 列出具体的要求和限制条件
4. **格式指定** - 说明期望的输出格式
5. **提供示例** - 如果可能，给出参考示例

### 进阶技巧
- **链式思考**: 使用"让我们一步步思考"提升推理能力
- **few-shot**: 提供多个示例帮助AI理解模式
- **角色扮演**: 通过角色设定提升回答的专业性
- **迭代优化**: 根据输出效果逐步调整提示词
- **组合策略**: 组合多个提示词以获得复杂功能
