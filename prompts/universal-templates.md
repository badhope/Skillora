## 🏗️ 万能架构

### 万能提示词公式

```
你是一位 {role}，具有 {expertise}。

我需要你帮助我完成以下任务：

任务描述：
{task_description}

具体要求：
1. {requirement_1}
2. {requirement_2}
3. {requirement_3}

请提供：
- {output_1}
- {output_2}
- {output_3}

输入信息：
{input_information}
```

### 扩展版架构

```
你是一位 {role}，拥有 {experience} 的经验。

背景信息：
{background_context}

主要任务：
{main_task}

具体要求：
- {req_1}
- {req_2}
- {req_3}

期望输出：
1. {output_1}
2. {output_2}
3. {output_3}

输入内容：
{input_content}

让我们一步步来思考：
```

## 📝 基础模板

### 1. 通用万能模板

```
你是一位 {role}，精通 {area}。

请帮我完成以下任务：

任务：
{task}

要求：
- {req1}
- {req2}
- {req3}

请提供高质量的输出。
```

### 2. 问题解决模板

```
你是一位 {role} 问题解决专家。

我遇到了以下问题：

问题描述：
{description}

已尝试的方法：
{attempts}

请帮我：
1. 分析问题原因
2. 提供解决方案
3. 给出实施步骤
4. 预防措施
```

### 3. 内容创作模板

```
你是一位专业的 {content_type} 创作者。

请帮我创作关于「{topic}」的内容：

创作目标：
{goal}

受众：
{audience}

风格要求：
{style}

长度要求：
{length}

请开始创作。
```

### 4. 学习辅导模板

```
你是一位耐心的 {subject} 老师。

我想学习「{topic}」：

我的水平：
{level}

我的目标：
{goal}

请提供：
1. 核心概念讲解
2. 学习方法建议
3. 练习题目
4. 资源推荐
```

## 🚀 高级模板

### 1. 深度分析模板

```
你是一位专业分析师，擅长深度分析。

请分析「{subject}」：

分析维度：
1. {dimension_1}
2. {dimension_2}
3. {dimension_3}

请提供：
- 全面分析
- 关键洞察
- 趋势判断
- 行动建议
```

### 2. 创意发散模板

```
你是一位创意大师，擅长头脑风暴。

请围绕「{theme}」进行创意发散：

创意目标：
{goal}

请提供：
1. 创意清单（至少10个）
2. 分类整理
3. 创意组合
4. 发展建议
```

### 3. 决策辅助模板

```
你是一位决策顾问，擅长决策分析。

我需要在以下选项中做决策：

选项：
- 选项A：{desc_a}
- 选项B：{desc_b}
- 选项C：{desc_c}

考量因素：
{factors}

请提供：
1. 各选项分析
2. 优缺点对比
3. 风险评估
4. 建议决策
5. 实施计划
```

### 4. 迭代优化模板

```
你是一位优化专家，擅长持续改进。

请优化「{subject}」：

当前状态：
{current_state}

优化目标：
{goals}

请提供：
1. 问题分析
2. 优化方案
3. 实施步骤
4. 效果评估
```

## 🛠️ 组合使用技巧

### 串联多个模板

```
第一步：[使用分析模板]
第二步：[使用创意模板]
第三步：[使用决策模板]
```

### 条件分支

```
如果情况是A，则：
[提示词A]

如果情况是B，则：
[提示词B]
```

### 链式思考

```
请先思考：
[思考问题1]

然后思考：
[思考问题2]

最后提供：
[期望输出]
```
