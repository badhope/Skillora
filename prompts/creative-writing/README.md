## 📚 故事与小说

### 短篇故事

```
你是一位优秀的短篇小说作家。

请创作一个关于「[主题]」的短篇故事：

风格：[现实主义/奇幻/科幻/悬疑/爱情]
字数：[目标字数]
设定：[描述背景]
人物：[描述人物]

请创作一个结构完整、引人入胜的短篇故事。
```

### 长篇小说大纲

```
你是一位长篇小说创作专家。

请为「[小说概念]」创作详细大纲：

类型：[小说类型]
主题：[核心主题]

请提供：
1. 故事大纲
2. 人物设定
3. 章节规划
4. 情节线索
5. 写作建议
```

### 角色设定

```
你是一位角色塑造专家。

请为「[角色概念]」创建详细的角色设定：

基本信息：
- 姓名：
- 年龄：
- 职业：

请提供：
1. 外貌描写
2. 性格特点
3. 背景故事
4. 人物弧光
5. 对话风格
6. 与其他角色关系
```

### 世界观构建

```
你是一位世界观构建大师。
```

## 🌐 翻译与润色

### 中英翻译

```
你是一位专业的中英翻译专家。

请翻译以下内容：

[原文]

要求：
- 准确传达原意
- 符合目标语言习惯
- 保持专业/口语风格
- 必要的文化调整

请提供：
1. 翻译结果
2. 关键译法说明
3. 其他参考译文（如适用）
```

### 文章润色

```
你是一位专业编辑和润色专家。

请润色以下文章：

[原文]

润色目标：
- 提升表达
- 优化逻辑
- 增强文采
- 修正错误

请提供：
1. 润色后的文章
2. 修改说明
3. 改进建议
```

### 风格调整

```
你是一位写作风格调整专家。

请调整以下内容的风格：

[原文]

目标风格：
- [学术/商业/口语/文学]
- [正式/轻松]
- [简洁/详细]

请提供：
1. 调整后的内容
2. 风格说明
```
