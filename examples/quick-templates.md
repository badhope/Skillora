# Quick Start Templates

Ready-to-use prompt templates for common tasks. Copy, adapt, and get started quickly!

## 📋 Usage

1. **Copy the template, fill in the placeholders with your specific information, and you're ready to go!

## 🎯 Quick Templates

### Template 1: Code Generation

```
You are an expert [programming language] developer with 10+ years of experience.

Task: Write a [function/class/app] to [detailed description]

Requirements:
- [ ] [requirement 1]
- [ ] [requirement 2]
- [ ] [requirement 3]

Please include:
- Clean, well-documented code
- Comments explaining key sections
- Example usage
- Notes on edge cases

Your code:
```

### Template 2: Content Writing

```
You are a professional [content_type] writer specializing in [topic_area].

Write a [content_type] about [topic]

Audience: [target_audience]
Tone: [formal/casual/persuasive/professional]
Length: [approximate_length]

Key points to include:
1. [point_1
2. [point_2
3. [point_3]

Guidelines:
- [guideline_1
- [guideline_2]
```

### Template 3: Data Analysis

```
You are an expert data analyst.

Data:
[paste_your_data_here]

Please analyze this data and provide:

1. Executive summary
2. Key insights
3. Trends & patterns
4. Recommendations
5. Actionable next steps

Format your response clearly with headings and bullet points.
```

### Template 4: Creative Writing

```
You are a creative storyteller and writer.

Write a [type_of_creative_work] with the following specifications:

Theme: [main_theme
Tone: [desired_tone]
Style: [writing_style]
Length: [approximate_length]

Key elements to include:
- [element_1
- [element_2
- [element_3]

Please structure:
```

### Template 5: Problem Solving

```
You are a problem-solving expert with experience in [domain].

Problem:
[describe_the_problem]

Please help by:
1. Understanding the problem
2. Analyzing it systematically
3. Exploring multiple approaches
4. Recommending the best solution
5. Explaining the reasoning

Think through this step by step.
```

### Template 6: Learning & Teaching

```
You are an expert teacher and educator.

I want to learn about [topic].

My current level: [beginner/intermediate/advanced]
Learning goal: [what_i_want_to_achieve]

Please:
1. Start with the basics
2. Build up complexity gradually
3. Use simple analogies
4. Provide practical examples
5. Give exercises or questions

Explain in simple terms.
```

### Template 7: Business & Strategy

```
You are a business strategist with expertise in [industry].

Situation:
[describe_business_situation]

Objective:
[what_we_want_to_accomplish]

Please provide:
1. Strategic analysis
2. Multiple approaches
3. Recommendations with pros/cons
4. Implementation roadmap
5. Risk assessment
```

## ⚡ Tips for Best Results

1. **Be specific**: The more details you provide, the better
2. **Define constraints**: Clear boundaries help focus the response
3. **Provide examples**: If helpful, include samples
4. **Iterate**: Start simple and refine as needed
