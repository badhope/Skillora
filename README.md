## 💡 提示词使用技巧

1. **明确角色** - 告诉 AI 扮演什么角色
2. **清晰任务** - 具体描述要完成的任务
3. **详细要求** - 列出具体的要求和期望
4. **格式明确** - 说明期望的输出格式
5. **提供示例** - 如果可能，给出好的示例
