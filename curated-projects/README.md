## 💡 贡献指南

欢迎提交新的优秀项目！请遵循以下步骤：

1. **选择合适的分类**
   - 开发框架 → `frameworks/`
   - 应用项目 → `applications/`

2. **创建 Markdown 文件**
   - 使用项目模板格式
   - 包含所有必需信息

3. **更新分类 README**
   - 在表格中添加新项目
   - 更新 Stars 数量

4. **提交 Pull Request**
