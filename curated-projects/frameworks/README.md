## 🏗️ 框架分类

### 单智能体框架
- **LangChain** - 最流行的通用 LLM 应用框架
- **LlamaIndex** - 专注于数据与 LLM 连接
- **Semantic Kernel** - 微软出品，企业级

### 多智能体框架
- **AutoGen** - 多智能体对话模式
- **CrewAI** - 角色化团队协作
- **MetaGPT** - 软件公司模式
- **OpenAI Swarm** - 轻量级交接
- **OpenAgents** - 协议级互操作
- **AgentScope** - 低门槛易用

### 状态与工作流
- **LangGraph** (LangChain 生态) - 图状态机
