## 简介

Dify 是一个一站式 AI 应用开发平台，提供从原型到生产的完整工作流。它支持可视化界面、提示词编排、知识库、多模型集成等功能，让开发者可以快速构建和部署 AI 应用，无需从零开始编写代码。

## 核心特点

- **可视化界面** - 友好的可视化界面，无需编程
- **提示词工程** - 强大的提示词编辑器和版本管理
- **知识库系统** - 支持文档上传、向量存储和语义检索
- **多模型支持** - 集成 OpenAI、Anthropic、Claude、Azure 等多种模型
- **API 管理** - 自动生成 API 接口
- **插件系统** - 丰富的插件生态
- **团队协作** - 支持团队协作和权限管理
- **多云部署** - 支持本地部署和云端部署

## 适用场景

- 企业内部 AI 助手

- 知识库问答系统
- 内容创作工具

- 无代码 AI 应用开发

## 快速开始

### 本地部署
```bash
# 使用 Docker Compose
git clone https://github.com/langgenius/dify.git
cd dify/docker
cp .env.example .env
docker compose up -d

# 访问 http://localhost
```

### 云服务
直接访问 [dify.ai](https://dify.ai) 注册使用

### 快速创建应用
1. 登录 Dify
2. 选择应用类型（聊天助手、文本生成等）
3. 配置提示词
4. 添加知识库（可选）
5. 发布为 API 或 Web 应用
