# Botpress

## 基本信息
- **官网**: https://botpress.com
- **GitHub**: https://github.com/botpress/botpress
- **类型**: 企业级对话式 AI 平台
- **维护者**: Botpress Inc.

## 简介

Botpress 是一个开源的企业级对话式 AI 平台，提供可视化对话流设计、意图识别、实体提取、对话管理等功能。它被全球超过 50 万开发者使用，是构建生产级对话系统的热门选择。

## 核心特点

### 可视化开发
- **Flow Builder** - 直观的可视化对话流设计
- **拖拽式节点** - 快速构建对话逻辑
- **实时预览** - 即时测试对话效果

### AI 能力
- **意图识别** - 内置 NLP 引擎
- **实体提取** - 智能实体识别
- **知识库集成** - 与外部知识库连接
- **多语言支持** - 支持 100+ 语言

### 部署灵活性
- **云端部署** - Botpress Cloud
- **自托管** - 完全控制数据
- **Docker 部署** - 一键容器化
- **多渠道** - Web、Slack、Discord、WhatsApp 等

### 企业功能
- **分析仪表盘** - 对话数据分析
- **团队协作** - 多用户工作区
- **版本控制** - 对话流程版本管理
- **SSO 支持** - 企业认证集成

## 快速开始

### Docker 部署
```bash
# 克隆仓库
git clone https://github.com/botpress/botpress.git
cd botpress

# 使用 Docker Compose 启动
docker compose up

# 访问 http://localhost:3000
```

### npm 部署
```bash
npm install -g @botpress/cli
bp init
bp start
```
