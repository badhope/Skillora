## 🔄 How to Sync & Download

### Clone Featured Repos

```bash
# Create a directory for external repos
mkdir -p external && cd external

# Clone key repositories
git clone https://github.com/dair-ai/Prompt-Engineering-Guide
git clone https://github.com/f/awesome-chatgpt-prompts
git clone https://github.com/PlexPt/awesome-chatgpt-prompts-zh
```

### Add as Submodules

```bash
git submodule add https://github.com/dair-ai/Prompt-Engineering-Guide external/prompt-engineering-guide
git submodule add https://github.com/f/awesome-chatgpt-prompts external/awesome-prompts
git submodule add https://github.com/PlexPt/awesome-chatgpt-prompts-zh external/awesome-prompts-zh
```

### Regular Sync

```bash
# Update all submodules
git submodule update --remote --merge
```
