# Workflow Agents
Business workflow automation agents.

## Projects

| Agent | Description |
|-------|-------------|
| Approval Workflow | Automated approval process |
| Onboarding | Employee onboarding automation |
| Invoice Processing | Automated invoice handling |

## Features

- Workflow definition language
- Human-in-the-loop support
- Audit trail
- Integration with enterprise systems

## Quick Start

```bash
cd workflow-agents/approval-workflow
npm install
npm run start
```

## Modules

- `approval-workflow/` - Approval automation
- `onboarding/` - Employee onboarding
- `invoice-processing/` - Invoice handling
