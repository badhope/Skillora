# Chat Agents
Conversational AI agents with memory and context management.

## Projects

| Agent | Description |
|-------|-------------|
| Personal Assistant | Your AI-powered personal assistant |
| Customer Support | AI customer support agent |
| Technical Support | Technical helpdesk agent |

## Features

- Context-aware conversations
- Memory management
- Multi-turn dialog support
- Personality customization

## Quick Start

```bash
cd chat-agents/personal-assistant
npm install
npm run start
```

## Modules

- `personal-assistant/` - Personal AI assistant
- `customer-support/` - Support agent
- `technical-support/` - Tech support agent
