## 🚀 快速开始

### 1. 聊天助手
```bash
cd ready-made-agents/chat-assistant
pip install -r requirements.txt
python main.py
```

### 2. 研究助手
```bash
cd ready-made-agents/research-agent
pip install -r requirements.txt
python main.py
```

### 3. 客服机器人
```bash
cd ready-made-agents/customer-support
pip install -r requirements.txt
python main.py
```

### 4. 任务自动化
```bash
cd ready-made-agents/task-automation
pip install -r requirements.txt
python main.py
```
