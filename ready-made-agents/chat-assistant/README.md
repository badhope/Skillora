# 🤖 Chat Assistant - 聊天助手

一个简单的多轮对话聊天助手，带记忆功能。

---

## 🚀 快速开始

### 安装
```bash
pip install -r requirements.txt
```

### 运行
```bash
python main.py
```

---

## 📁 目录结构

```
chat-assistant/
├── main.py           # 主程序
├── requirements.txt  # 依赖
└── README.md         # 说明
```

---

## 💡 功能特点

- ✅ 多轮对话记忆
- ✅ 简单的命令系统
- ✅ 友好的交互界面
- ✅ 可配置的 LLM

---

## 🛠️ 使用说明

运行 `main.py` 后，你可以：
- 直接对话：`你好！`
- 查看历史：`/history`
- 清空对话：`/clear`
- 退出：`/quit`
