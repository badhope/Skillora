# ⚡ Task Automation - 任务自动化

一个能够自动分解和执行复杂任务的 AI 助手。

---

## 🚀 快速开始

### 安装
```bash
pip install -r requirements.txt
```

### 运行
```bash
python main.py
```

---

## 📁 目录结构

```
task-automation/
├── main.py             # 主程序
├── planner.py          # 任务规划器
├── executor.py         # 任务执行器
├── requirements.txt    # 依赖
└── README.md           # 说明
```

---

## 💡 功能特点

- ✅ 任务自动分解
- ✅ 步骤规划
- ✅ 任务执行追踪
- ✅ 执行日志记录
- ✅ 进度可视化

---

## 🛠️ 使用说明

运行 `main.py` 后，你可以：
- 描述任务：`帮我写一篇关于 AI 的文章`
- 查看计划：`/plan`
- 执行计划：`/execute`
- 查看进度：`/progress`
- 退出：`/quit`
