- **语言**: TypeScript / Python
- **维护者**: Reworkd

## 简介

AgentGPT 是一个在浏览器端部署的 AI 代理，用户可以在浏览器中直接使用，无需搭建开发环境。它让用户可以给 AI 一个目标，然后 AI 会自动规划和执行任务来实现目标。特别适合非技术用户使用。

## 核心特点

- **浏览器端部署** - 完全在浏览器中运行，无需服务器
- **直观界面** - 简单易用的可视化界面
- **无需编程** - 非技术用户也可以轻松使用
- **目标导向** - 设置目标，AI 自动执行
- **任务可视化** - 可视化展示任务执行过程
- **工具集成** - 支持搜索、网页浏览等工具
- **部署简单** - 一键部署到 Vercel 等平台

## 适用场景

- 快速任务自动化
- 个人助理应用
- 学习和教育工具
- 无需编程的 AI 代理部署
- 客户服务机器人定制
- 原型验证和演示

## 快速开始

### 在线使用
直接访问 https://agentgpt.reworkd.ai

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/reworkd/AgentGPT.git
cd AgentGPT

# 安装依赖
npm install

# 配置环境
cp .env.example .env
# 编辑 .env 文件

# 运行
npm run dev

# 访问 http://localhost:3000
```

### Vercel 部署
```bash
# 点击 GitHub 仓库的 Deploy to Vercel 按钮
# 或使用 Vercel CLI
vercel
```
