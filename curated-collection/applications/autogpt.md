## 简介

AutoGPT 是最早和最著名的自主 AI 代理项目之一，它通过将大语言模型与工具调用、记忆系统、推理规划等能力相结合，实现了从被动问答系统向主动任务执行系统的重要转变。

## 核心特点

- **高度自主** - 能够自主规划和执行任务，无需人工干预
- **多工具集成** - 支持网络搜索、文件操作、代码执行等多种工具
- **记忆系统** - 短期和长期记忆管理
- **目标导向** - 根据设定的目标自动规划步骤
- **Web 界面** - 提供友好的用户界面
- **插件系统** - 支持第三方插件扩展功能

## 适用场景

- 自动化研究和数据收集
- 内容创作和编辑
- 项目管理和任务规划
- 学习和知识探索
- 编程辅助和代码生成
- 多步骤复杂任务自动化

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/Significant-Gravitas/AutoGPT.git
cd AutoGPT

# 安装依赖
cd autogpt
pip install -r requirements.txt

# 配置环境
cp .env.template .env
# 编辑 .env 文件，添加 OpenAI API 密钥

# 运行
python -m autogpt
```

### Docker 运行
```bash
docker compose run --rm auto-gpt
```
