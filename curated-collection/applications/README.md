## 🎯 应用分类

### 自主代理
- **AutoGPT** - 高度自主的任务执行代理
- **AgentGPT** - 浏览器端部署的自主代理

### 开发平台（无代码/低代码）
- **Dify** - 开源自托管
- **Coze** - 字节跳动出品，国内首选
- **Botpress** - 企业级开源对话系统

### 语音助手
- **Voiceflow** - Alexa/Google Assistant 首选

---

## 💡 选择指南

### 快速上手
| 需求 | 推荐平台 | 理由 |
|------|---------|------|
| 国内用户 | Coze | 中文友好，渠道丰富 |
| 国外用户 | Dify | 开源活跃，社区支持 |
| 完全免费 | AgentGPT | 浏览器直接用 |
| 企业自部署 | Dify / Botpress | 开源可控 |

### 功能需求
| 需求 | 推荐平台 | 特点 |
|------|---------|------|
| 知识库 | Dify / Coze | 内置 RAG |
| 多渠道 | Coze / Botpress | 微信、飞书等 |
| 语音 | Voiceflow | Alexa/Google 原生 |
| 复杂工作流 | Dify | 可视化编排 |

---

## 💡 项目特点

### 自主代理
- 能够自主规划和执行任务
- 支持工具调用和互联网搜索
- 提供用户界面

### 开发平台
- 低代码/无代码开发
- 可视化界面
- 一键部署功能
- 集成多种 LLM

### 对话系统
- 意图识别
- 实体提取
- 多轮对话
- 知识库集成
