# Hermes Agent

## 基本信息
- **GitHub**: https://github.com/NousResearch/hermes-agent
- **⭐ Stars**: 119k+
- **语言**: Python
- **维护者**: NousResearch

## 简介

Hermes Agent 是一个持续学习的智能体框架，能够不断从对话中学习和进化。它支持多种模式，包括聊天、代码执行、文件操作等。

## 核心特点

### 持续学习
- **记忆系统**: 长期记忆和短期记忆
- **学习机制**: 从对话中自动学习
- **进化能力**: 不断改进自身

### 多模态支持
- **代码执行**: 支持 Python 代码
- **文件操作**: 读写文件
- **网络搜索**: 实时信息获取

### 扩展性
- **插件系统**: 支持自定义插件
- **多 LLM 支持**: 支持多种模型

## 适用场景

- 个人助手
- 学习伴侣
- 代码助手
- 研究助手

## 快速开始

```bash
git clone https://github.com/NousResearch/hermes-agent.git
cd hermes-agent
pip install -r requirements.txt
```

```python
from hermes import HermesAgent

agent = HermesAgent()
response = agent.chat("你好！")
print(response)
```
